Form

An HTML form is used to collect user input. The user input is most often sent to a server for processing.

<form> <input type="text" placeholder="Enter name"> </form>

HTML headings are titles or subtitles that you want to display on a webpage.

HTML <a> Tag

The <a> tag defines a hyperlink, which is used to link from one page to another.

CSS

Selectors

Selectors define which HTML elements are styled.

p { color: black; }
This text is black

Grid Layout

Grid allows creating complex layouts easily.

display: grid

Background color

The background-color property specifies the background color of an element.

background-color: lightblue;

Border

The border-style property specifies what kind of border to display.

{ border-style: dashed; }

Protocols

HTTP & HTTPS

HTTP (HyperText Transfer Protocol) is the foundation of communication for the World Wide Web. HTTPS is the secure version that uses SSL/TLS encryption.

Response Status Codes:

  • 1xx - Informational responses
  • 2xx - Success (e.g., 200 OK)
  • 3xx - Redirection (e.g., 301 Moved Permanently)
  • 4xx - Client errors (e.g., 404 Not Found)
  • 5xx - Server errors (e.g., 500 Internal Server Error)

DNS

DNS (Domain Name System) translates domain names into IP addresses.

Common DNS Record Types:

  • A - Maps a domain to an IPv4 address
  • AAAA - Maps a domain to an IPv6 address
  • CNAME - Alias for another domain
  • MX - Mail exchange record
  • TXT - Stores text information (e.g., SPF records for email authentication)

IP Address

An IP address identifies a device on the internet.

Finding an IP Address:

ping google.com

or use an online tool like:

nslookup google.com
